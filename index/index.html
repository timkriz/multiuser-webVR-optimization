WebVR Optimization Experiments

  1. ROOM 1: 3D MODEL COMPLEXITY
    A-frame VR room to measure effect of number of triangles on performance.
    1. COMPLEX 75.860 triangles
    2. OPTIMIZED 862 triangles
  2. ROOM 2: OBJECT COUNT
    A-frame VR room to measure effect of object count / draw calls on performance.
    1. COMPLEX Scene with 1000 geometries X 588 triangles.
    2. OPTIMIZED Scene with merged 1 geometry x 588000 triangles.
  3. ROOM 3: LIGHTNING and SHADERS
    A-frame VR room to measure effect of lightmapping on performance.
    1. COMPLEX Physically based rendering (PBR).
    2. OPTIMIZED Lights baked into textures - lightmapping.
  4. ROOM 4: LATENCY
    A-frame VR room to measure effect of latency on UX.
    1. WEBSOCKETS Networked A-frame - Websockets.
    2. WEBRTC Networked A-frame - WebRTC.
